HTB: Arkham

Arkham是一个中等难度的靶机，但是它的难度可以和困难相媲美。其中涉及了lucks解密、JSF ViewState反序列化、ost邮件分析、UAC绕过等相关知识。ViewState反序列化漏洞让我学到了很多，虽然其中的数据是加密的，但是它提供了一个用于执行攻击的密钥使得我能够成功获取shell，上线后在电子邮件中找到了管理员密码，需要绕过UAC限制拿到最后的flag。感兴趣的同学可以在HackTheBox中进行学习。

HTB: Devel

Devel是一个容易的靶机，其中涉及了ftp匿名登录、内核提权等知识。有意思的点是该靶机的ftp目录同时也是webroot目录，攻击者可以直接上传webshell来进行远程控制。感兴趣的同学可以在HackTheBox中进行学习。

Posted on 2021-12-18HackTheBox

HTB: Optimum

Optimum是一个容易的靶机，其中涉及了HTTP File Server远程代码执行利用、MS16-032内核提权等知识。HTTP File Server的远程代码执行漏洞演示了可通过非常简单的方式获取shell，但是也有需要注意的地方，即反弹的meterpreter要与系统位数保持一致。感兴趣的同学可以在HackTheBox中进行学习。

Posted on 2021-12-14HackTheBox

HTB: Netmon

Netmon是一个容易的靶机，其中涉及了ftp匿名登录、PRTG网络监视器命令注入、psexec使用等知识。PRTG网络监视器的命令注入非常有趣，存在许多种方式对目标站点进行命令操作，与此同时它还是少见的未获取shell就可以拿到flag的靶机。感兴趣的同学可以在HackTheBox中进行学习。

Posted on 2021-12-13HackTheBox

HTB: Hackback

Hackback是一个非常难的靶机，其中涉及了钓鱼网站、参数模糊测试、日志中毒、HTTP代理、WinRm服务、UserLogger服务、NFTS数据流、恶意dll等知识，总共耗费了我五天时间来攻克和研究，感兴趣的同学可以在HackTheBox中进行学习。我觉得主要难的还是日志相关的那两个点，这是不是和近期log4j2远程代码执行有异曲同工之妙呢？让我们一起来看看。
