web.config上传绕过

web.config文件是一个xml文件，主要用于存储ASP.NET Web应用程序的配置信息，使用.NET创建一个Web应用程序后，默认情况下会在根目录生成一个默认的web.config文件，包括默认的配置设置，所有的子目录都继承它的配置设置。如果你想修改子目录的配置设置，你可以在该子目录下新建一个Web.config文件。它可以提供除从父目录继承的配置信息以外的配置信息，也可以重写或修改父目录中定义的设置。

web.config 类似于 Apache 中 .htaccess 文件，主要包含 IIS 服务器上的 Web 应用程序的设置和配置数据，我们可以根据下面这篇文章上传 web.config 来运行 asp 代码

参考文章：https://soroush.secproject.com/blog/tag/unrestricted-file-upload/
