HTB: Optimum

Optimum是一个容易的靶机，知识点涉及HTTP File Server远程代码执行利用、MS16-032内核提权等。其中HTTP File Server存在远程代码执行漏洞可直接获取权限，但是需要注意的是反弹的meterpreter要与系统位数保持一致。感兴趣的同学可以在HackTheBox中进行学习。

0x01 侦查

端口探测

首先通过nmap对目标进行端口扫描


可以发现目标只开放了80端口并识别该系统为Windows

80端口


该界面是 Http File Server 的 2.3 版本，主要用于上传和分享文件。

漏洞查找

使用 Searchsploit 来探测 HFS 的相关漏洞，可以发现在 2.3 这个版本中存在远程代码执行漏洞
