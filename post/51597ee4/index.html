0x01 侦查

端口探测

扫描结果显示目前目标开放了80、135、446、5985、49669端口

密码破解

点击包含其中的附件，我们发现配置信息中存在账号和密码

分别是类型为 Cisco 7 和 Cisco 5 的哈希值

Cisco Type 5

使用 john 对 Type 5 哈希进行爆破


成功破解发现密码为stealth1agent

Cisco Type 7

针对 Type 7 的哈希则可以借助在线网站进行破解
破解网站：https://www.ifm.net.nz/cookbooks/passwordcracker.html



解密后密码为rout3r/$uperP@ssword、admin/Q4)sJu\Y8qz*A3?d

Cisco Type 7解密脚本

Type 7 存在静态加密密钥tfd;kfoA,.iyewrkldJKD以及弱算法加密。可直接编写 python 脚本进行破解

smb枚举

有了对应的账号密码那么我们就可以对 smb 服务进行爆破，使用 crackmapexec 对其进行爆破

爆破成功，使用账号密码hazard/stealth1agent查询smb共享

当前用户只能读取$IPC，$IPC主要用于进程通信的共享，通常会出现空连接会话，但是目前我们需要凭证才能读取

由此可知通过暴力破解 sid 可获取用户列表

0x03 权限提升[administrator]

Firefox获取凭证

我们注意到在该靶机上还存在 Firefox 浏览器，说明其中可能存在利用点

查看进程发现 Firefox 正在运行中，进程中可能包含存储的明文密码

为了下载火狐浏览器进程，我们可以上传进程读取工具procdump.exe至目录C:\Windows\System32\spool\drivers\color下，当然也可以使用 Powersploit 中的Out-Minidump.ps1工具，虽然两者效果一样，但是我更倾向于使用前者

借助 procdump 将 Firefox 中的进程从内存中 dump 并保存

下载导出的 Firefox 进程

0x04 总结

使用访客用户 guest 可登陆目标网站。后台中存在信息泄漏，在泄漏的Cisco配置文件中包含相关账号密码信息，其中密码是Cisco特有的Tyoe 5以及Type 7类型的将哈希值。将其成功破解后拿到明文枚举smb服务，成功枚举出账号密码。

通过枚举出来的账号密码登陆IPC，登录后成功拿到系统用户的SID，使用SID爆破技术可获取用户列表，将其整理为字典后枚举WinRM服务，成功后登陆用户并拿到用户权限。

靶机中存在Firefox进程，将其转储后可在其中找到管理员密码，最终我们使用该密码通过WinRM登陆管理员账户。
