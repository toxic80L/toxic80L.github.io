HTB: Bastion

Bastion是一个简单的靶机，知识点涉及smb匿名登录漏洞、vhd镜像挂载和提取mRemoteNG密码等。其中并没有涉及许多漏洞知识，而是仅靠凭证获取和密码破解即可完成靶机攻克。感兴趣的同学可以在HackTheBox中进行学习。

0x01 侦查

端口探测

首先使用nmap进行端口扫描



扫描结果显示目标开放了22、135、139、445、5985等端口

445端口

由于目标存在 smb 匿名登录漏洞，我们使用 smbclient 查看默认共享


目前只能进入 Backups 目录

vhd镜像挂载

发现 note.txt 并通过命令下载


查看 note.txt 提示我们VPN比较慢不要将其转换到本地

在\WindowsImageBackup\L4mpje-PC\Backup 2019-02-22 124351\目录下发现备份相关文件

直接进入该目录并在其中可以看到 vhd 文件（vhd是系统备份文件）

如果尝试下载就会如 note.txt 中所说的将会话费很长时间，因此可以挂载镜像


其中这里的 qemu-nbd 是 qemu 中的一个用户态工具，它可以使用-r、-c命令将VHD 文件挂载到本地

0x02 上线[l4mpje]

注册表转储哈希并破解

进入~/hackthebox/Bastion/mnt/目录，查看vhd镜像文件


在Windows/System32/config目录下存在 SYSTEM 和 SAM 文件，这其中存在系统的凭证信息


将这两个文件复制到本地目录下


使用 samdump2 或 secretdump.py 可获取凭证信息，其中存在 L4mpje 的哈希


在解密网站中对其密码进行解密，发现 L4mpje 的密码为 bureaulampje
解密网站：https://crackstation.net/

ssh登录

通过之前扫描到的 ssh 端口来登录目标系统

在当前用户的桌面上成功拿到第一个flag

0x03 权限提升[administrator]

提取密码

C:\Users\L4mpje\AppData\APPDATA目录下存在一个用于管理远程连接的应用程序 mRemoteNG，而 confCons.xml 中则存放着其所有连接信息和凭据


通过 scp 传输 confCons.xml

在本地打开文件发现管理员用户名和加密后的密码，我们需要借助相关脚本对 mRemoteNG 的加密值进行解密
脚本地址：https://github.com/kmahyyg/mremoteng-decrypt/blob/master/mremoteng_decrypt.py


成功拿到密码为thXLHM96BeKL0ER2

管理员登录

通过 ssh 登录目标系统


在管理员桌面上成功拿到第二个flag
