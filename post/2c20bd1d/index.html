HTB: Access

Access是一个简单的靶机，知识点涉及Access数据库信息收集、telnet登陆、lnk文件分析、runas提权、dpapi信用提权等。匿名登陆ftp后在其中找到一个Access数据库，通过信息收集拿到账号密码并使用telnet登陆，存在lnk文件，分析后发现可借助runas提权或使用dpapi直接提权至管理员权限。感兴趣的同学可以在HackTheBox中进行学习。

0x01 侦查

端口探测

首先通过nmap对目标进行端口扫描


扫描结果显示目标开放了21、23以及80端口

80端口

访问80端口，是一个机房图片

使用 gobuster 对80端口进行目录扫描


但是在目录扫描并没有发现什么可利用点。

21端口

由于目标站点存在 ftp 匿名登录漏洞，使用匿名用户anonymous/anonymous尝试登录ftp


标存在两个目录：/Backups和/Engineer，/Backups目录下存在数据库backup.mdb，与此同时在/Engineer目录下存在文件Access Control.zip

将它们下载至本地目录


0x02 上线[security]

数据库信息收集

首先查看 mdb 数据库的文件类型，确认为 Access 数据库

安装 mdbtools 用于打开 Access 数据库，也可以在 windows 中利用 Access 客户端

mdbtools 中包含多个命令，能够查询数据库版本、查询数据表、导出数据等

首先使用mdb-tables列出该数据库的数据表


但是未查看成功，查找原因后发现原文件大小为5652480，而下载下来的文件大小只有5651666。

将 ftp 模式设置为二进制传输并下载


对比文件大小后发现文件下载成功并再次查看数据库表


其中包含多个数据库表，使用mdb-export转储acc_antiback和acc_door表中的数据


通过 bash 对其中的表进行循环遍历来查看各个表中的数据长度


查询auth_user表发现其中存在用户信息


成功拿到如下账号密码

解压zip文件

使用unzip解压Access Control.zip


但是结果显示不支持，尝试通过7z来进行解压，如果未安装可以使用以下命令来安装

需要输入密码，尝试使用从数据库获取到的两个密码，最终通过密码access4u@security成功解压

查看解压后的Access Control.pst的文件类型，发现该文件是一个 outlook 电子邮件


使用 readpst 可以将其转换为 mbox 格式

而 mbox 文件可以直接通过 cat 命令进行查看，但是为了提升阅读体验，我们使用 mutt 来打开该文件。其中参数-R代表只读，-f代表标识从中读取的文件

信件内容包含账号密码为security/4Cc3ssC0ntr0ller
